# 📌 AI-Powered Image Captioning Platform  
منصة ويب لتوليد وصف تلقائي للصور باستخدام نماذج ذكاء اصطناعي محلية.

---

## 🚀 Overview  
**AI-Powered Image Captioning Platform** هي منصة ويب متكاملة تقوم بتحليل الصور وتوليد وصف نصي دقيق لها باستخدام نماذج رؤية حاسوبية (Computer Vision) ونماذج لغة طبيعية (NLP).  
تعمل المنصة **بشكل كامل دون الحاجة لأي اتصال بالإنترنت**، مما يجعلها مناسبة للبيئات ذات القيود التقنية أو الأمنية.

تعتمد المنصة على:

- نموذج **Vision Transformer (BLIP)** لتوليد الوصف باللغة الإنجليزية  
- نموذج **Argos Translate** للترجمة المحلية إلى العربية  
- واجهة ويب مبنية باستخدام **Flask**  
- نظام إدارة ملفات وصور بسيط وسريع  
- لوحة تحكم (Dashboard) لعرض الإحصائيات وتحليل الأداء  

---

## 📡 How It Works

1. يقوم المستخدم برفع صورة عبر واجهة الويب  
2. يتم تمرير الصورة إلى نموذج **BLIP**  
3. النموذج يولّد وصفًا باللغة الإنجليزية  
4. يتم تمررير الوصف إلى **Argos Translate** للترجمة المحلية  
5. يتم عرض النتائج في الواجهة  
6. يتم حفظ السجلات في ملف JSON لتحليل الأداء لاحقًا
